# Portfolio

A personal portfolio website built with React and Vite.

## Overview

This project is a modern single-page portfolio built using:

- React
- Vite
- Framer Motion
- React Router DOM
- React Icons
- Particles background effects using `react-tsparticles`

## Live Demo

Hosted at: https://lalithamanasvini.github.io/portfolio

## Features

- Responsive landing page
- Animated UI transitions
- Projects section
- Navigation bar component
- Particle background effect

## Project Structure

- `index.html` - main HTML entry point
- `src/main.jsx` - app bootstrapping
- `src/App.jsx` - root component
- `src/components/` - reusable UI components
- `src/pages/` - page-level views
- `src/index.css` / `src/App.css` - global and app styles

## Getting Started

### Prerequisites

- Node.js 18+ recommended
- npm

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project uses `gh-pages` for GitHub Pages deployment.

```bash
npm run deploy
```

## Notes

- The `homepage` field in `package.json` is set to the GitHub Pages URL.
- Update the project metadata and content as needed for your personal portfolio.
